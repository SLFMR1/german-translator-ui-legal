# Glossary — Untranslatable and Domain Terms

## Universal Rules (Apply to Any Project)

### Keep in English (Standard in Swiss Tech/Business)

These are standard vocabulary in Swiss-German tech contexts — do NOT germanize them:

Agent, AI, API, Business, Compliance, CRM, Dashboard, Deals, ERP, Inbox, Integration, Meeting, Onboarding, Portfolio, SAP, Smart Contract, Support, Token, Tokenization, Tools, Workflow

**Note on E-Mail:** "E-Mail" IS the standard German spelling (hyphenated, capitalized). It's not an English loanword kept as-is — it's the germanized form. Never write "Email" (no hyphen) or "email" (lowercase) in German text.

### Context-Dependent UI Terms

These English words translate differently depending on UI context. Using the wrong one sounds broken to native speakers.

| English | UI Context | German | Wrong alternative (and why) |
|---------|-----------|--------|---------------------------|
| toggle | Nav menu button (hamburger) | Menü öffnen | "umschalten" — implies settings toggle |
| toggle | Settings switch (on/off) | umschalten / aktivieren | "öffnen" — implies opening something |
| close | Dismiss dialog/modal | Schliessen | "Nah" — means nearby |
| close | End account/session | Beenden / Kündigen | "Schliessen" — means shutting a window |
| save | Store data/file | Speichern | "Sparen" — means saving money |
| save | Discount/money | Sparen | "Speichern" — means storing data |
| share | Send content to others | Teilen | "Anteil" — means ownership share |
| share | Equity/ownership | Anteil | "Teilen" — means distributing |
| view | See/display content | Anzeigen / Ansehen | "Aussicht" — means scenic view |
| view | Page view (analytics) | Seitenaufruf | "Ansicht" — means visual layout |
| check | Verify/validate | Prüfen | "Scheck" — means bank cheque |
| check | Checkbox/tick | Markieren / Ankreuzen | "Prüfen" — means verifying |
| set up | Configure/initialize | Einrichten | "Aufstellen" — means physically placing |
